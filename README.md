# InvoiceForge

Desktop-Programm zum Erstellen und Speichern von Rechnungen als PDF.

## Was ist dieses Programm?

InvoiceForge ist ein Programm für Ihren Rechner, mit dem Sie Rechnungen selbst erstellen und als PDF-Datei speichern können. Sie tragen Ihre Firmendaten einmal ein und können sie für alle Rechnungen wiederverwenden. Alle Daten bleiben auf Ihrem Computer – Sie brauchen dafür kein Konto im Internet und keine Cloud.

## Was kann das Programm?

- **Firmendaten speichern und wiederverwenden:** Name, Adresse und weitere Angaben zu Ihrem Unternehmen werden gespeichert und bei jeder neuen Rechnung automatisch übernommen.
- **Rechnungen anlegen:** Sie erfassen Kundendaten, Rechnungsnummer, Rechnungsdatum und Fälligkeitsdatum in einem übersichtlichen Formular.
- **Positionen verwalten:** Sie fügen Posten (Positionen) hinzu oder entfernen sie. Zwischensumme, Steuer und Gesamtbetrag werden automatisch berechnet.
- **Rechnung als PDF speichern:** Über eine Schaltfläche öffnet sich ein Speicher-Dialog; Sie wählen den Ordner und Dateinamen und speichern die Rechnung als PDF-Datei auf Ihrem Rechner.

## Typische Anwendungsfälle

- Rechnungen für Kunden nach Auftragsabschluss ausstellen
- Wiederkehrende Rechnungen mit bereits gespeicherten Firmendaten erstellen
- Rechnungen ohne Internetverbindung schreiben (z. B. vor Ort beim Kunden)
- Kleine Abrechnungen mit mehreren Positionen anlegen
- PDF-Rechnung zum Versand per E-Mail oder zum Drucken erzeugen

## Installation

1. Laden Sie den Installer von der [Releases-Seite des Projekts](https://github.com/deadfrogface/InvoiceForge/releases) herunter.
2. Starten Sie die heruntergeladene Datei und folgen Sie den Anweisungen des Setups.
3. Öffnen Sie das Programm über das Startmenü oder die angelegte Verknüpfung.

## Nutzung

- **Firmendaten:** Geben Sie Ihre Firmendaten ein und speichern Sie sie über die Funktion „Firmendaten speichern“. Sie werden bei jeder neuen Rechnung automatisch verwendet.
- **Rechnung ausfüllen:** Tragen Sie Kundendaten, Rechnungsnummer, Datum und Fälligkeit in den dafür vorgesehenen Feldern ein.
- **Positionen:** Fügen Sie Posten hinzu (z. B. Bezeichnung, Menge, Preis). Positionen können Sie bei Bedarf wieder entfernen. Zwischensumme, Steuer und Gesamtbetrag werden automatisch berechnet.
- **PDF erstellen:** Mit „PDF erstellen“ öffnet sich ein Speicher-Dialog. Wählen Sie den gewünschten Ordner und Dateinamen; die Rechnung wird als PDF-Datei auf Ihrem Rechner gespeichert.

## Systemanforderungen

- Windows 10 oder neuer

## Offline-Hinweis

Das Programm funktioniert vollständig offline. Es ist keine Internetverbindung erforderlich.
